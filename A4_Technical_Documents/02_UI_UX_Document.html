UI/UX Document

CreConnect — Influencer–Brand Collaboration Platform
Frontend: React 18 (Vite) Routing: React Router v6 Styling: Tailwind CSS Date: June 2026

1. Design Principles

PrincipleImplementation
Role-based navigationThree distinct sidebar layouts: Brand, Creator, Admin — each with role-specific menu items
Dark-first themeDefault dark mode (#0b0a12 background, indigo #6d5cff accent) with light mode toggle
Consistent cardsAll data uses glassmorphism-style cards with var(--surface) and var(--border) CSS tokens
Micro-interactionsFramer Motion animations on page enter (.page-enter), button presses, and modal reveals
Real-time awarenessSocket.io notification badges and live message threads update without page refresh
Mobile responsiveSidebar collapses at 768 px; cards switch to single-column grid on mobile

2. Public Screens

Public / Unauthenticated

2.1 Landing Page

Landing Page — / Hero section with tagline, platform stats (12K+ creators, 800+ brands), feature highlights, AI matching demo, pricing tiers, and testimonials.

2.2 Role Selection & Sign-Up

Role Selection — /signup User selects Creator or Brand before entering registration details.
Creator Sign-Up — /signup/creator Fields: Display Name, Username, Email, Password. Social platform links optional.
Brand Sign-Up — /signup/brand Fields: Company Name, Contact Name, Industry (dropdown), Email, Password.
Login — /login Creator/Brand role toggle, email + password fields, Google OAuth, server status indicator.
Forgot Password — /forgot-password Email input triggers OTP email. OTP entered on next step to unlock password reset.

3. Brand Screens

Brand Role
Brand Dashboard — /brand/dashboard KPI cards (Total Campaigns, Active Collaborations, Pending Offers, Messages Waiting), Revenue chart, Audience Reach, and AI-recommended creators section.
Campaigns — /brand/campaigns List of all campaigns with status badges (Draft / Published). Create new campaign button opens a detailed form.
Discover Creators — /brand/search AI Match and Saved toggle buttons. Niche filter chips (Fashion, Gaming, Food, Tech, Beauty, Lifestyle, Travel). Search bar with keyword suggestions.
Collaborations — /brand/collaborations Tracks all creator partnerships with status indicators.
Messages — /brand/messages Real-time Socket.io chat threads with creators. New messages appear without page refresh.
Notifications — /brand/notifications System alerts: new applications, messages, collaboration updates.
Settings — /brand/settings Profile edit (company name, logo, industry), password change, notification preferences.
Activity Log — /brand/activity Timestamped log of all brand account actions and campaign events.

4. Creator Screens

Creator Role
Creator Dashboard — /creator/dashboard KPI cards (Followers, Engagement Rate, Active Campaigns, Pending Offers), Earnings chart, Quick Actions panel, and New Offers list.
Profile — /creator/profile Bio, niche tags, platform handles (Instagram/TikTok/YouTube), and media portfolio upload.
Browse Campaigns — /creator/campaigns Discover open brand campaigns. Filter by niche, platform, and budget range. Click to apply.
Collaborations — /creator/collaborations Track all applications and active collabs: Pending → Active → Completed pipeline.
Messages — /creator/messages Real-time chat with brands. Unread threads marked with indicator dot.
Find Brands — /creator/find-brands Browse brand profiles filtered by industry and budget range.
Reviews — /creator/reviews Star ratings and written reviews received from brands after completed collaborations.
Notifications — /creator/notifications Alerts for new offers, application decisions, messages, and system events.

5. Key User Flows

Flow 1 — Brand Posts a Campaign

Landing Page → [Get Started] → Role Selection → [Brand] → Brand Sign-Up Form → Email OTP Verification → Admin Approval → Login → Brand Dashboard → [+ New Campaign] → Campaign Form (title, niche, budget, deliverables) → [Publish] → Campaign live in creator browse feed

Flow 2 — Creator Applies to Campaign

Login → Creator Dashboard → [Browse Campaigns] → Campaign listing with filters → [Apply] → Pitch note textarea → Submit → Status: "Pending" in Collaborations tab → Brand accepts → Status: "Active"

Flow 3 — Real-Time Messaging

Brand selects creator → [Send Collab Request] → Creator receives Socket.io push notification → Creator opens Messages → new thread visible → Both parties exchange messages in real time → Brand marks collaboration complete → Review prompt
